# AI 视频创作工具 - H5 Demo

基于 React + Vite 的 AI 视频生成工具前端 Demo。

## 功能

- **模板选择页**：5 种类型风格、10 个模板卡片、分类筛选和搜索
- **视频创作页**：提示词输入、多语言翻译（8 种语言）、参考图片上传、画面比例/分辨率/时长选择
- **视频生成**：模拟 AI 视频生成过程（进度条动画）
- **预览与下载**：视频播放预览、全屏查看、下载

## 技术栈

- React 18
- Vite 8
- 纯前端，无需后端
